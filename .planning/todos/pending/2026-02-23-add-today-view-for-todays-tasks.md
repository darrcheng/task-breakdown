## Problem

Users need a focused view that shows only tasks scheduled for today. Currently there's no way to quickly see just today's tasks without scanning the full calendar or list view. A dedicated "Today" view would reduce cognitive load and help users focus on what matters right now.

## Solution

- Add a "Today" view/tab alongside existing calendar and list views
- Filter tasks to only show items with today's date
- Display in a clean, focused layout optimized for daily task management
- Include task status, category, and quick-action controls
