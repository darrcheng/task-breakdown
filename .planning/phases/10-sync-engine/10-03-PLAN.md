    2. Implement `migrateLocalData(uid: string)` in `src/firebase/sync.ts`:

       a. Add module-level state: `let migrating = false`

       c. Set `migrating = true` at start, `false` at end (in finally block).

       d. Read all local data:
          ```
          const localTasks = await db.tasks.toArray();
          const localCategories = await db.categories.toArray();
          ```

       e. If all arrays are empty, return early (fresh device, nothing to upload).

       f. Check what already exists in Firestore (for each of the 3 collections):
          ```
          const existingTaskDocs = await getDocs(collection(firestore, `users/${uid}/tasks`));

          ```

       g. Build list of records to upload (local records whose String(id) is NOT in existing IDs):
          ```
          const tasksToUpload = localTasks.filter(t => !existingTaskIds.has(String(t.id)));
          const categoriesToUpload = localCategories.filter(c => !existingCatIds.has(String(c.id)));
          const aiSettingsToUpload = localAiSettings.filter(a => !existingAiIds.has(String(a.id)));
          ```

       h. Batch write in chunks of 450:
          ```
          const allOps = [
            ...tasksToUpload.map(t => ({ table: 'tasks', record: t })),
            ...categoriesToUpload.map(c => ({ table: 'categories', record: c })),
            ...aiSettingsToUpload.map(a => ({ table: 'aiSettings', record: a })),
          ];

          for (let i = 0; i < allOps.length; i += 450) {
            const chunk = allOps.slice(i, i + 450);
            const batch = writeBatch(firestore);
            for (const op of chunk) {
              const ref = doc(collection(firestore, `users/${uid}/${op.table}`), String(op.record.id));
              batch.set(ref, serializeForFirestore(op.record, op.record.id!));
            }
            await batch.commit();
          }
          ```

       i. Note: cloud-only records (exist in Firestore but not locally) will arrive via onSnapshot listeners that are started AFTER migration in Plan 04. No need to pull them here.
