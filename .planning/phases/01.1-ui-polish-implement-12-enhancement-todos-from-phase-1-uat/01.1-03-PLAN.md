**TaskInlineCreate.tsx — Category selector (Todo #10):**

Add a compact category `<select>` dropdown below the title input. Use a simple select here (not the full combobox from Plan 02) because inline create needs to be compact and fast.

1. Import `useCategories` from hooks and add `useState` for categoryId:
```tsx
import { useState, useRef, useEffect } from 'react';
import { db } from '../../db/database';
import { useCategories } from '../../db/hooks';
import { CATEGORY_ICONS } from '../../utils/categories';
```

2. Add state and load categories:
```tsx
const [categoryId, setCategoryId] = useState<number>(0);
const categories = useCategories();

// Set default category when categories load
useEffect(() => {
  if (categoryId === 0 && categories && categories.length > 0 && categories[0].id) {
    setCategoryId(categories[0].id);
  }
}, [categories, categoryId]);
```

3. Update `handleSubmit` to use the selected categoryId instead of fetching the default:
```tsx
const handleSubmit = async (e: React.FormEvent) => {
  e.preventDefault();
  if (!title.trim()) return;

  const finalCategoryId = categoryId || (categories?.[0]?.id ?? 1);

  await db.tasks.add({
    title: title.trim(),
    description: '',
    date,
    status: 'todo',
    categoryId: finalCategoryId,
    createdAt: new Date(),
    updatedAt: new Date(),
  });

  setTitle('');
  inputRef.current?.focus();
};
```

Remove the old `const categories = await db.categories.toArray(); const defaultCategoryId = categories[0]?.id ?? 1;` from inside handleSubmit — we now use the reactive hook.

4. Add the category select below the title input in the form JSX. Render both in a flex row so the form stays compact:

```tsx
<form onSubmit={handleSubmit} className="px-4 py-2">
  <div className="flex gap-2">
    <input
      ref={inputRef}
      type="text"
      value={title}
      onChange={(e) => setTitle(e.target.value)}
      onKeyDown={handleKeyDown}
      placeholder="Add a task... (Enter to create, Esc to close)"
      className="flex-1 px-3 py-2 border border-blue-300 rounded-md text-sm focus:ring-2 focus:ring-blue-400 focus:border-blue-400 outline-none bg-blue-50/50"
    />
    <select
      value={categoryId}
      onChange={(e) => setCategoryId(Number(e.target.value))}
      className="px-2 py-2 border border-blue-300 rounded-md text-sm focus:ring-2 focus:ring-blue-400 focus:border-blue-400 outline-none bg-blue-50/50 max-w-[140px]"
    >
      {categories?.map((cat) => (
        <option key={cat.id} value={cat.id}>
          {cat.name}
        </option>
      ))}
    </select>
  </div>
</form>
```

The select is compact (max-w-[140px]) and visually matches the input styling (blue-300 border, bg-blue-50/50). This keeps inline create fast while adding category choice.
