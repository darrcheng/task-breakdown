Create a new component file with a searchable category dropdown. Key behaviors:
- Text input that filters categories by name (case-insensitive substring match)
- When not focused/open, displays the selected category name
- On focus, opens a dropdown list of matching categories
- Each list item shows the category icon (via CATEGORY_ICONS) + name
- If the typed text does not exactly match any category name, show a "Create [query]" option at the bottom with a Plus icon
- Clicking "Create" adds a new category to the database (icon: 'folder', isDefault: false) and selects it
- Recently-selected categories appear first. Track recent IDs in localStorage under key `'taskbreaker-recent-categories'` (max 3 entries). On category select, prepend its ID. Sort display: recent IDs first, then alphabetical.
- Accessibility: `role="combobox"` on input, `aria-expanded`, `role="listbox"` on dropdown, `role="option"` on items
- Use `onMouseDown` (not `onClick`) on list items to prevent input blur from closing the dropdown before selection registers
- Use `onBlur` with `setTimeout(150ms)` to close the dropdown (standard React pattern for blur-before-mousedown)
- Keyboard: ArrowDown/Up to navigate options, Enter to select highlighted option, Escape to close

const RECENT_KEY = 'taskbreaker-recent-categories';
const MAX_RECENT = 3;

function getRecentIds(): number[] {
  try {
    const stored = localStorage.getItem(RECENT_KEY);
    return stored ? JSON.parse(stored) : [];
  } catch {
    return [];
  }
}

function saveRecentId(id: number) {
  const recent = getRecentIds().filter(r => r !== id);
  recent.unshift(id);
  localStorage.setItem(RECENT_KEY, JSON.stringify(recent.slice(0, MAX_RECENT)));
}

The component exports `CategoryCombobox` as a named export. Full implementation includes:
