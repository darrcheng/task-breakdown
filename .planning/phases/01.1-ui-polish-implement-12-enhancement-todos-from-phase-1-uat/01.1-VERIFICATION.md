Plans 09 and 10 summary:

Plan 09 implemented three targeted fixes from the UAT recheck. `TaskListItem.tsx` adds `text-green-600` and `decoration-green-600` to the departing state with explicit child-element overrides on icon, title, and badge (necessary because child elements with their own `colors.text` class would otherwise override the parent's green color). `CategoryCombobox.tsx` adds a `handleClear` callback and an absolutely-positioned X button with `pr-8` input padding. `DayGroup.tsx` replaces the text `+` glyph with a `<Plus />` SVG icon from lucide-react and removes the text-metric classes that were only relevant for text glyphs.
