Create three new files:

**src/firebase/auth.ts** — Firebase auth helpers:
- Import `getAuth`, `GoogleAuthProvider`, `signInWithPopup`, `signInWithRedirect`, `getRedirectResult`, `signOut` from `firebase/auth`
- Import `app` from `./config`
- Export `auth = getAuth(app)`
- Export `googleProvider = new GoogleAuthProvider()`
- Export `redirectResultPromise = getRedirectResult(auth).catch(...)` — call once on module load to handle redirect results. Log errors with `console.error('Redirect sign-in error:', error.code, error.message)`. Return `null` on error.
- Export `async function signInWithGoogle(useRedirect: boolean): Promise<void>` — if `useRedirect`, call `signInWithRedirect(auth, googleProvider)`, else call `signInWithPopup(auth, googleProvider)`
- Export `async function signOutUser(): Promise<void>` — call Firebase `signOut(auth)`

**src/contexts/AuthContext.tsx** — React context for auth state:
- Interface `AuthState { user: User | null; loading: boolean }` (User from `firebase/auth`)
- Create context with default `{ user: null, loading: true }`
- `AuthProvider` component: uses `useState<AuthState>` starting at `{ user: null, loading: true }`. In `useEffect`, subscribe to `onAuthStateChanged(auth, (user) => setState({ user, loading: false }))`. Return unsubscribe on cleanup.
- Export `useAuth()` hook that returns `useContext(AuthContext)`

**src/components/ui/AuthLoadingScreen.tsx** — Loading screen per user spec:

- Shows "taskpad" text: `text-2xl font-semibold text-slate-800`
- After 2 seconds, a small spinner fades in below the text using `useState` + `useEffect` with `setTimeout(2000)`
- Use `Loader2` from `lucide-react` with `animate-spin` class for the spinner
- Spinner fade-in: wrap in a div with `transition-opacity duration-300` toggling between `opacity-0` and `opacity-100`
