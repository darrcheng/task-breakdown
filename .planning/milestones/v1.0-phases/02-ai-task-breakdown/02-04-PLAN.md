A component that displays subtasks of a parent task inside the task modal:
- Props: `parentId: number`, `parentDepth: number`, `onOpenSubtask: (task: Task) => void`
- Uses `useSubtasks(parentId)` hook for live data
- Renders each subtask as a row with:
  - Left indent based on depth: `ml-${(subtask.depth - parentDepth) * 4}` or equivalent padding (4px per level)
  - Status checkbox (circle indicator matching existing status cycling pattern — slate for todo, amber for in-progress, emerald for done)
  - Task title (clickable to open subtask in modal via onOpenSubtask callback)
  - Category icon (small, from existing CATEGORY_ICONS mapping)

- Empty state: "No subtasks yet" with a subtle text
- Section header: "Subtasks ({count})" with a small Lucide `ListTree` icon

**src/components/task/ParentBadge.tsx:**

- Shows: small pill with `ListTree` icon + count (e.g., "3")
- Tailwind: `inline-flex items-center gap-0.5 text-xs text-slate-500 bg-slate-100 rounded-full px-1.5 py-0.5`
- Should be very lightweight — just an icon + number, no click handler (parent modal shows full subtask list)
