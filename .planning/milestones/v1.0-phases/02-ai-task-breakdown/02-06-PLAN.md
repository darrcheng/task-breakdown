Fix the two-part data-flow bug in TaskModal.tsx that causes blank modals and stale status display.

**Part A — Replace useEffect-based viewingTask sync with direct derivation:**

Currently (lines 24, 31-37):
```tsx
const [viewingTask, setViewingTask] = useState<Task | undefined>(task);
useEffect(() => {
  setViewingTask(task);
  setParentStack([]);
  ...
}, [isOpen, task]);
```

Replace with a navigation-override pattern:
```tsx
const [navigationOverride, setNavigationOverride] = useState<Task | undefined>();
const [parentStack, setParentStack] = useState<Task[]>([]);
```

Derive viewingTask directly (no state, no useEffect delay):
```tsx
const viewingTask = navigationOverride ?? task;
```

Update the reset useEffect to only clear navigation state (not set viewingTask):
```tsx
useEffect(() => {
  setNavigationOverride(undefined);
  setParentStack([]);
  if (!isOpen) {
    breakdown.cancelBreakdown();
  }
}, [isOpen, task]);
```

Update `handleOpenSubtask` to use `setNavigationOverride(subtask)` instead of `setViewingTask(subtask)`.

Update `handleBackToParent` to use `setNavigationOverride(parent)` instead of `setViewingTask(parent)`. When popping the last item from parentStack (going back to the root task), set `setNavigationOverride(undefined)` so the original `task` prop takes over.

**Part B — Add key prop to TaskForm:**

On the TaskForm JSX (currently line 171), add a key prop:
```tsx
<TaskForm
  key={currentTask?.id ?? 'new'}
  initialData={currentTask}
  ...
/>
```

This forces React to unmount/remount TaskForm whenever the task identity changes, causing useState initializers to re-run with fresh data.

**Part C — Add useLiveQuery for parent task and fix breadcrumb:**

```tsx

  () => (currentTask?.parentId ? db.tasks.get(currentTask.parentId) : undefined),
  [currentTask?.parentId],
);
```


Update `handleBackToParent` to handle the case where parentStack is empty but parentTask exists (direct open from calendar). const handleBackToParent = () => {
```tsx
  if (parentStack.length > 0) {

    setNavigationOverride(parent);

  }

};
```
