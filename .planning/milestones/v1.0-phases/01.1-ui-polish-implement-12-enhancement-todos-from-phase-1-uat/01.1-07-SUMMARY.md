## Decisions Made

- Deleted the auto-select useEffect entirely (rather than guarding with `initialData?.categoryId`) — the state initializer `useState(initialData?.categoryId ?? 0)` already correctly handles edit mode. The useEffect was only needed if categories loaded asynchronously after mount, but CategoryCombobox handles its own category data via useLiveQuery.
- Used an IIFE pattern `{!isOpen && selectedCategory && (() => { ... })()}` to render the icon conditionally, keeping icon logic colocated with the input rather than extracting to a variable.
- Added `flex-1` to the task list div so blank space below the last task card is also part of the clickable area.
