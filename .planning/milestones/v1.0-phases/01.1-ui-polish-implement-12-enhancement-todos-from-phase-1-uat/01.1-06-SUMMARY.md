## Decisions Made

- Used `departing` local state + `setTimeout(1500ms)` to delay DB write so the Dexie live query doesn't immediately re-fire and remove the task while animation plays
- Removed the two-click confirmReopen pattern entirely (no amber pulsing ring, no "click again to reopen" tooltip) as it was unwanted UX
- `leading-none` is the minimal fix for + centering - removes implicit line-height that was offsetting the glyph from flexbox center
- `flex-[N]` proportional approach is more robust than fixed widths as it scales with container
