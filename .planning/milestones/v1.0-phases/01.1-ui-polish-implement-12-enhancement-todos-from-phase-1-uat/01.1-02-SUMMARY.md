**Clickable status pill buttons and searchable category combobox with create-from-search and recent-first ordering**

## Accomplishments
- Status selection uses three colored clickable pill buttons instead of a dropdown
- Category selection uses a searchable combobox with icon display, type-to-filter, and keyboard navigation
- Users can create new categories inline by typing a name that doesn't exist
- Recently-selected categories appear first in the list (tracked in localStorage)
