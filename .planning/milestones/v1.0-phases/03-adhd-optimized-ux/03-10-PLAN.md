**TaskCard.tsx — Add subtask count badge:**

Import `useSubtasks` from `../../db/hooks` and `ListTree` from lucide-react.

Add a subtask count indicator. Use the useSubtasks hook with the task's ID to get subtask count. Only render if there are subtasks:

```tsx
const subtasks = useSubtasks(task.id ?? 0);
const subtaskCount = subtasks?.length ?? 0;
const subtaskDoneCount = subtasks?.filter(s => s.status === 'done').length ?? 0;
```

Render after the estimate badge and before ParentBadge:
```tsx
{subtaskCount > 0 && (
  <span className="flex items-center gap-0.5 text-[10px] text-slate-400 flex-shrink-0">
    <ListTree className="w-3 h-3" />
    {subtaskDoneCount}/{subtaskCount}
  </span>
)}
```

NOTE: useSubtasks is a Dexie liveQuery hook — it will reactively update when subtasks change. The task.id check (task.id ?? 0) prevents querying with undefined. If task.id is 0, useSubtasks returns empty array which is fine.
