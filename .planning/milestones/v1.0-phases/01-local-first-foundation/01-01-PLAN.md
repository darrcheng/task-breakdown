  <action>
    **src/types/index.ts** — Define core types:
    ```typescript
    export type TaskStatus = 'todo' | 'in-progress' | 'done';
    export type ViewMode = 'calendar' | 'list';
    export type CalendarView = 'month' | 'week';

    export interface Task {
      id?: number;
      title: string;
      description: string;
      date: string;            // 'YYYY-MM-DD' format, local timezone
      status: TaskStatus;
      categoryId: number;
      createdAt: Date;
      updatedAt: Date;
    }

    export interface Category {
      id?: number;
      name: string;
      icon: string;           // lucide-react icon name
      isDefault: boolean;
    }
    ```

    **src/db/database.ts** — Dexie database with schema and seed data:
    - Create Dexie instance named 'TaskBreaker'
    - Version 1 stores: tasks with indexes on `++id, date, status, categoryId`; categories with `++id, name`
    - Use `db.on('populate')` to seed 5 default categories: Work (briefcase), Personal (user), Health (heart), Learning (book-open), Errands (shopping-cart)
    - Export the `db` instance

    **src/db/hooks.ts** — Reactive query hooks using `useLiveQuery`:
    - `useTasksByDate(date: string, showCompleted: boolean)`: Returns tasks for a specific date, filtering out 'done' status unless showCompleted is true
    - `useTasksByDateRange(startDate: string, endDate: string, showCompleted: boolean)`: Returns tasks for a date range (inclusive), for calendar month view
    - `useCategories()`: Returns all categories
    - `useCategoryMap()`: Returns a Map<number, Category> for O(1) lookup by ID
    - All hooks return `undefined` while loading (standard useLiveQuery behavior)

    **src/utils/dates.ts** — Calendar date utilities using date-fns:
    - `getCalendarDays(month: Date): Date[]` — Returns all days to display in a calendar grid (from start-of-week of month start to end-of-week of month end, typically 35 or 42 days)
    - `getWeekDays(date: Date): Date[]` — Returns 7 days for the week containing the given date
    - `formatDateKey(date: Date): string` — Formats to 'yyyy-MM-dd' using date-fns `format()` (local timezone)
    - `isToday(date: Date): boolean` — Re-export from date-fns
    - `isSameMonth(date: Date, month: Date): boolean` — Re-export from date-fns
    - `getDaysInRange(start: Date, end: Date): Date[]` — Returns all dates in range (for list view)

    **src/utils/categories.ts** — Category constants:
    - `DEFAULT_CATEGORIES`: Array of default category objects matching the seed data
    - `CATEGORY_ICONS`: Record mapping icon string names to lucide-react component references
    - `STATUS_COLORS`: Record mapping TaskStatus to Tailwind class objects { bg, border, text } using an ADHD-friendly accessible palette:
      - todo: slate-100/slate-300/slate-700 (neutral, calm)
      - in-progress: amber-50/amber-300/amber-800 (warm, active)
      - done: emerald-50/emerald-300/emerald-700 (positive, complete)

    IMPORTANT: Store dates as 'YYYY-MM-DD' strings (not Date objects) in IndexedDB. Use `format(date, 'yyyy-MM-dd')` from date-fns which uses local timezone. Never parse date-only strings with `new Date()` directly.
