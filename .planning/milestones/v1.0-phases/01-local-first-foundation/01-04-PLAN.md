    - Dialog centered, max-w-md, rounded corners, subtle shadow, semi-transparent backdrop

    - After creation, clears input and stays open for rapid entry
    - Esc to dismiss
    - Props: `date: string`, `onClose: () => void`

    **src/components/task/TaskInlineEdit.tsx** — Inline expand for list view task editing:
    - Expands below/replaces the TaskListItem when clicked
    - Contains the full TaskForm (title, status, category, description)
    - Props: `task: Task`, `onClose: () => void`
    - On submit: `db.tasks.update(task.id, {..., updatedAt: new Date()})`
    - On delete: `db.tasks.delete(task.id)`
    - On cancel/close: collapse back to TaskListItem
    - Smooth expand/collapse animation via Tailwind transition
