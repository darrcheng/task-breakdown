  <name>Task 1: Install Firebase SDK, create config and infrastructure files</name>
  <files>package.json, src/firebase/config.ts, .env.example, firebase.json, .firebaserc, firestore.rules, .gitignore</files>
  <action>
1. Install dependencies:
   ```bash
   npm install firebase@^12.10.0
   npm install -D firebase-tools@^15.9.0
   ```

2. Create `src/firebase/config.ts` with minimal Phase 8 initialization:
   ```typescript
   import { initializeApp } from 'firebase/app';
   import { getFirestore } from 'firebase/firestore';

   const firebaseConfig = {
     apiKey: import.meta.env.VITE_FIREBASE_API_KEY,
     authDomain: import.meta.env.VITE_FIREBASE_AUTH_DOMAIN,
     projectId: import.meta.env.VITE_FIREBASE_PROJECT_ID,
     storageBucket: import.meta.env.VITE_FIREBASE_STORAGE_BUCKET,
     messagingSenderId: import.meta.env.VITE_FIREBASE_MESSAGING_SENDER_ID,
     appId: import.meta.env.VITE_FIREBASE_APP_ID,
   };

   if (!firebaseConfig.projectId) {
     throw new Error(
       'Firebase config missing. Copy .env.example to .env.local and fill in your Firebase project values.'
     );
   }

   export const app = initializeApp(firebaseConfig);
   export const firestore = getFirestore(app);
   ```
   Use `firestore` as export name (not `db`) to avoid shadowing the Dexie export in `src/db/database.ts`. Use `getFirestore` not `initializeFirestore` — persistence config is deferred to Phase 10. Do NOT import `firebase/auth` — that is Phase 9 scope.

3. Create `.env.example` with all 6 VITE_FIREBASE_* variable placeholders, including a comment block explaining these are public client keys and security comes from Firestore rules. Use the exact template from 08-RESEARCH.md Code Examples section.

4. Create `firestore.rules` with deny-all rules:
   ```
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /{document=**} {
         allow read, write: if false;
       }
     }
   }
   ```

5. Create `firebase.json`:
   ```json
   {
     "firestore": {
       "rules": "firestore.rules"
     }
   }
   ```

6. Create `.firebaserc`:
   ```json
   {
     "projects": {
       "default": "taskpad"
     }
   }
   ```

7. Add to `.gitignore` (append at end):
   ```
   # Firebase
   .firebase/
   firebase-debug.log
   ```
   Note: `.env.local` is already covered by the existing `*.local` pattern.
  </action>
