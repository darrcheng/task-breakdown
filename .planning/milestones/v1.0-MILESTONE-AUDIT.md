- `BottomTabBar` has 3 tabs: Calendar, List, Settings. No Someday tab.

- Affects: PLAT-02, ADHD-04

**2. Swipe-complete bypasses celebration** (HIGH)

- Also missing `hapticFeedback` call on swipe path
- Affects: ADHD-03, TASK-07

**3. Inline create skips time estimate** (LOW)
- `TaskInlineCreate` calls `db.tasks.add()` without `triggerEstimate`
- Time estimate only fires from `TaskModal.handleSubmit`
- Affects: ADHD-02
